# 01_Business - 사업 관련 문서

## 목적
사업 전략, 시장 분석, 비즈니스 모델 관련 핵심 문서를 보관합니다.

## 포함 문서
- Executive Summary (사업 개요서)
- Market Analysis (시장 분석 보고서)
- Business Model Canvas (비즈니스 모델 캔버스)
- ROI Analysis (투자 수익 분석)

## 작성 순서
1. Executive Summary (최우선)
2. Market Analysis
3. Business Model Canvas
4. ROI Analysis

## 문서 작성 가이드
- 비즈니스 용어는 구체적인 수치와 함께 설명
- 시장 데이터는 출처를 명확히 표기
- 경쟁사 분석은 객관적 데이터 기반
