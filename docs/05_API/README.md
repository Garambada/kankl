# 05_API - API 명세서

## 목적
시스템 간 통신 인터페이스를 명확하게 정의합니다.

## 포함 문서
- REST API Specification (REST API 명세서)
- API Endpoints (API 엔드포인트 정의)
- Request/Response Examples (요청/응답 예시)
- Error Codes (에러 코드 정의)

- RESTful: REST 아키텍처 원칙 준수
- Versioning: API 버전 관리 (/v1/, /v2/)
- Documentation: Swagger/OpenAPI 사용
