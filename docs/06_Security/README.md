# 06_Security - 보안 및 리스크 관리

## 목적
시스템과 데이터의 보안을 보장하고 리스크를 관리합니다.

## 포함 문서
- Security Architecture (보안 아키텍처)
- Risk Management Plan (리스크 관리 계획)
- Compliance Checklist (컴플라이언스 체크리스트)
- Incident Response Plan (사고 대응 계획)

## 보안 원칙
- Defense in Depth: 다층 보안
- Zero Trust: 모든 접근 검증
- Privacy by Design: 설계 단계부터 프라이버시 고려
